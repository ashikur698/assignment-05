Important FAQ

The DOM is like a big map of a web page. It helps the computer understand the page's structure and allows you to change things on the page using code.

You can select an element by its name, like getting a book from a shelf by its title. For example, you can use special words in JavaScript like document.getElementById().

Event delegation is when you attach a single event listener to a parent element instead of adding one to each child. It makes your code simpler and can improve performance.

You can change an element's attributes and styles using JavaScript. For example, element.style.color = 'red'; changes the text color to red.
